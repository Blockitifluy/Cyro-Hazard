# Cyro Hazard

## Descripion

Cyro Hazard is an infinitily generated zombie game, where you survive in a cold wasteland in West Texas.

Zombies follow the rules of evolution were they mutate based on the environment. Special weather such as: Blood Moons, Snow Storms; posing challages for surivors and zombies alike.

## Features

- Abondoned cities and buildings,
- Heavy Weather,
- Zombie mutation,
- Realistic Crafting Tree

## Installing the Source Code

Make sure to download [Godot Mono](https://godotengine.org/download) (The C Sharp version of Godot), before cloning the source code.

```bash
git clone https://github.com/Blockitifluy/Cyro-Hazard
```
